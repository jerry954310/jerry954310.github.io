專案經驗

Gourping! 您的揪團好選擇

Android studio

專案簡介

Gourping! 為一款社群應用程式，目的在增進中正大學學生的互動效率，參考現有FB社團實作出買賣、團購、共乘三大主題板塊
學生在平台上註冊帳號後即可在三大功能區內發文及留言，達成即時的互動性

開發動機

現有FB社團互動方式較不方便，學生必須花費時間查找不同的社團來完成互動，且貼文常與動態混合出現，因此需要一款整合性的應用程式

個人貢獻

將 UI 設計圖轉化為 Xml 程式碼，並使用 Java 開發 App 前端，完善使用者體驗

專案簡報

NBA購票決策系統

R studio

專案簡介

NBA購票管道琳瑯滿目，又有不同方案可以選擇，本專案旨在不同的出席率下，模擬出購買季票及單場票的平均價格，協助使用者作出成本最低的決策
本專案假設購票管道分為季票及單場票，季票價格固定，單場票則是受當天對戰組合及雙方明星球員出賽數影響，本專案由5000次模擬結果做出結論

機率分配使用

使用者到場率: triangular distribution
使用者到場狀況: exponential distribution
明星球員出席率: triangular distribution
明星球員出席狀況: binomial distribution

數據收集、程式邏輯發想、程式模擬結果

探討消費者的環保意識及健康意識對於綠色餐廳的消費行為之影響

Smart PLS / SPSS

研究動機

台灣於80年代起開始推行綠色消費風潮，呼籲消費者透過綠色消費來減少環境污染以及提升環境品質，近年來也有越來越多商家和團體響應綠色消費的概念。 在綠色消費概念中，綠色餐廳的重要性日漸顯著，關於綠色餐廳的研究文獻也日漸增多，但其中卻只有少數文獻探討消費者在綠色餐廳的消費行為，因此我們想透過此研究探討會影響消費者去綠色餐廳消費的因素

研究缺口

過往研究中，消費者的環保意識和健康意識，被認為和綠色消費行為之間沒有絕對的關係，本研究著重環保意識、健康意識是否會影響綠色餐廳消費行為，同時加入社會壓力程度、社群媒體壓力及性傾向為衡量依據

研究方法

參考相關文獻、設計問卷並投放，回收150份有效問卷，並使用統計軟體驗證假說

個人貢獻

文獻資料收集、問卷結果編碼、操作統計軟體得出研究結果
